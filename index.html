Ionic & Cordova

Cordova

  • Framework Open-Source pour le devéloppement d'application multiplateforme
  • Fondation Apache
  • Basé sur les Webviews
  • HTML, CSS et JavaScript
  • Permet l'accès aux composants physiques (Boussole, GPS, Stockage ...) selon la plateforme

Plateformes supportées

Ionic

  • Basé sur Cordova
  • Développement accès sur Angular
  • Ajout de composants UI (bouton, liste, menu ...) pré-designé
  • Plateforme supportée : Android, iOS, Windows

Ionic

Ionic v1 / v2

  • Angular JS 1.4.x / Angular 2
  • JavaScript / TypeScript
  • Support officiel de Windows

Inconvénients des Webviews

  • Problèmes liés au développement web (compatibilité du CSS notamment)
  • Gestion verticale de la page (alignement, répartition du contenu)
  • Accès parfois limité aux composants physiques
  • Performances pas toujours optimales
  • Windows : IE 11 Enjoy =D
