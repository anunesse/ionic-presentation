# Ionic & hybrid frameworks

### Client ionic (clickmeme)
#### Requirements
* Node : 6.9.2 (npm)
* Android : Android SDK, $ANDROID_HOME
* IOS : XCode, ios-deploy
* npm install -g cordova ionic

#### Install
* cd clickmeme/
* npm install
* ionic platform add android (or ios)
* ionic state restore
* ionic run android (or ios)
